# 03. 画面設計

## WP管理画面のメニュー構成

```
SnapBaton（トップメニュー）
  ├── グループ一覧
  ├── 画像一覧（全グループ横断）
  ├── 投稿セット一覧
  ├── ゴミ箱
  └── 設定
```

## 画面一覧

### グループ一覧画面
- グループをカード形式で表示
- タグでフィルタリング
- 各カードにサムネイル（代表画像）・グループ名・画像枚数・作成日
- 新規グループ作成ボタン

### グループ詳細画面
- グループ名・説明の表示・編集
- グループタグの編集（タグ入力UIコンポーネント）
- 画像一覧（グリッド/リスト切替）
- 画像の複数アップロード（ドラッグ&ドロップ対応）
- 画像の並べ替え（ドラッグ&ドロップ）
- 各画像のタイトル・説明文はインライン編集（フォーカス外れで自動保存）
- 各画像にタグ入力UIを表示
- このグループに関連する投稿セット一覧

### 画像編集（エディタ）
- サムネイルクリックで `@fujiruki/react-image-editor-lightbox` が起動
- テキストオーバーレイ追加・移動・編集
- 矢印描画・回転
- トリミング
- ピンチズーム（モバイル対応）
- 保存でWPメディアライブラリの画像を上書き更新

### 画像アクションボタン（グループ詳細画面内）
各画像カードに以下のアクションボタンを表示:
- コピー（説明文をクリップボードにコピー）
- クリア（履歴に保存してからタイトル・説明文をクリア）
- 共有URL（AI連携用URLをコピー）
- ダウンロード（オリジナルサイズ）
- ゴミ箱（ソフトデリート、canDelete権限時のみ表示）

### 投稿セット作成・編集画面
- 投稿先SNS選択（X / Instagram / Facebook / note / その他）
- 画像選択エリア（グループから選択、複数可）
- 選択画像の順番入れ替え（ドラッグ&ドロップ）
- 投稿文入力エリア
- クリップボードコピーボタン群（投稿文・共有URL）
- ステータス（下書き / 投稿済み）
- 保存ボタン

### 投稿セット一覧画面
- テーブル形式で一覧表示
- 投稿先SNS・ステータス・作成日でフィルタ
- 各行にサムネイル・投稿文（冒頭）・投稿先・ステータス

### ゴミ箱画面
- 削除済みグループ・画像をテーブル形式で一覧
- サムネイル・名前・種類（画像/グループ）・元のグループ名・削除日を表示
- 「復元」ボタンで元のグループに戻す（グループ復元時は中の画像も一括復元）

### 設定画面
- プラグインの基本設定
- サムネイルサイズ設定
- 権限のカスタマイズ（将来的）

### スマホ公開アップロード画面（WP管理画面外）
- WPログイン不要の独立ページ
- Step1: 4桁パスコード入力
- Step2: グループ選択（既存一覧 or 新規作成）
- Step3: 画像・動画の複数選択アップロード（プログレス表示）
- Step4: 完了メッセージ（「続けてアップロード」ボタン付き）
- モバイルファーストのシンプルUI
- SnapBatonのロゴ・タイトル付き
- 「ホーム画面に追加」の案内バナー

### 公開ギャラリー（サイト訪問者向け）
- `[snapbaton_gallery]` ショートコードで固定ページに埋め込み
- `is_public=1` のグループのみ表示
- マソンリーグリッド（CSS columns）でレスポンシブ表示
- タグフィルタボタン（ワンクリックで即時絞り込み）
- ライトボックス:
  - 画像を画面中央に表示領域の90%幅で表示
  - 画像の下にタイトル（白文字・太字）
  - その下に説明文（グレー文字・スクロール可能）
  - 左右矢印 / スワイプ / キーボードで画像送り
  - 背景クリック or ×ボタンで閉じる
  - 動画はライトボックス内でインライン再生

## 表示モード

| モード | 説明 |
|:--|:--|
| グリッド表示 | サムネイルが並ぶ。各画像の下にタイトル。一覧性重視 |
| リスト表示 | 横一列。サムネイル+タイトル+説明文が並ぶ。詳細確認向け |
