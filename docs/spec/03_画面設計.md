# 03. 画面設計

## WP管理画面のメニュー構成

```
SnapBaton（トップメニュー）
  ├── グループ一覧
  ├── 画像一覧（全グループ横断）
  ├── 投稿セット一覧
  ├── ゴミ箱
  └── 設定
```

## 画面一覧

### グループ一覧画面
- グループをカード形式で表示
- タグでフィルタリング
- 各カードにサムネイル（代表画像）・グループ名・画像枚数・作成日
- 新規グループ作成ボタン

### グループ詳細画面
- グループ名・説明の表示・編集
- 画像一覧（グリッド/リスト切替）
- 画像の複数アップロード（ドラッグ&ドロップ対応）
- 画像の並べ替え（ドラッグ&ドロップ）
- このグループに関連する投稿セット一覧

### 画像詳細画面
- 画像の大きい表示
- タイトル・説明文の入力フォーム
- EXIFメタデータ表示
- クリップボードコピーボタン群（説明文・画像・共有URL）
- オリジナルサイズダウンロードボタン
- この画像の投稿履歴一覧
- 画像エディタ起動ボタン

### 投稿セット作成・編集画面
- 投稿先SNS選択（X / Instagram / Facebook / note / その他）
- 画像選択エリア（グループから選択、複数可）
- 選択画像の順番入れ替え（ドラッグ&ドロップ）
- 投稿文入力エリア
- クリップボードコピーボタン群（投稿文・共有URL）
- ステータス（下書き / 投稿済み）
- 保存ボタン

### 投稿セット一覧画面
- テーブル形式で一覧表示
- 投稿先SNS・ステータス・作成日でフィルタ
- 各行にサムネイル・投稿文（冒頭）・投稿先・ステータス

### ゴミ箱画面
- 削除済みグループ・画像を一覧
- 復元・完全削除ボタン

### 設定画面
- プラグインの基本設定
- サムネイルサイズ設定
- 権限のカスタマイズ（将来的）

## 表示モード

| モード | 説明 |
|:--|:--|
| グリッド表示 | サムネイルが並ぶ。各画像の下にタイトル。一覧性重視 |
| リスト表示 | 横一列。サムネイル+タイトル+説明文が並ぶ。詳細確認向け |
