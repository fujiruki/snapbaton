# 04. データ設計

## テーブル構成

WPプラグインとしてカスタムテーブルを使用。
テーブル名にはWPのプレフィックス + `snapbaton_` を付ける。
（例: `wp_snapbaton_groups`）

### wp_snapbaton_groups（グループ）

| カラム | 型 | 説明 |
|:--|:--|:--|
| id | BIGINT UNSIGNED AUTO_INCREMENT | PK |
| name | VARCHAR(255) | グループ名 |
| description | TEXT | グループの説明 |
| author_id | BIGINT UNSIGNED | 作成者のWPユーザーID |
| created_at | DATETIME | 作成日時 |
| updated_at | DATETIME | 更新日時 |
| deleted_at | DATETIME NULL | ゴミ箱移動日時（NULLなら有効） |

### wp_snapbaton_tags（タグ）

| カラム | 型 | 説明 |
|:--|:--|:--|
| id | BIGINT UNSIGNED AUTO_INCREMENT | PK |
| name | VARCHAR(100) | タグ名 |

### wp_snapbaton_group_tags（グループ×タグ）

| カラム | 型 | 説明 |
|:--|:--|:--|
| group_id | BIGINT UNSIGNED | FK → groups.id |
| tag_id | BIGINT UNSIGNED | FK → tags.id |

### wp_snapbaton_images（画像）

| カラム | 型 | 説明 |
|:--|:--|:--|
| id | BIGINT UNSIGNED AUTO_INCREMENT | PK |
| group_id | BIGINT UNSIGNED | FK → groups.id |
| attachment_id | BIGINT UNSIGNED | WPメディアライブラリのattachment ID |
| title | VARCHAR(255) | タイトル |
| description | TEXT | 説明文 |
| sort_order | INT | グループ内の表示順 |
| author_id | BIGINT UNSIGNED | アップロード者のWPユーザーID |
| created_at | DATETIME | アップロード日時 |
| updated_at | DATETIME | 更新日時 |
| deleted_at | DATETIME NULL | ゴミ箱移動日時 |

※ 画像ファイル自体はWPメディアライブラリに保存（attachment_idで参照）。
  オリジナルサイズを保持するため、WP側の画像圧縮設定に注意。

### wp_snapbaton_image_tags（画像×タグ）

| カラム | 型 | 説明 |
|:--|:--|:--|
| image_id | BIGINT UNSIGNED | FK → images.id |
| tag_id | BIGINT UNSIGNED | FK → tags.id |

PK: (image_id, tag_id)

※ グループタグ（group_tags）と同じ `tags` マスタテーブルを共有する。

### wp_snapbaton_post_sets（投稿セット）

| カラム | 型 | 説明 |
|:--|:--|:--|
| id | BIGINT UNSIGNED AUTO_INCREMENT | PK |
| title | VARCHAR(255) | 投稿セットの管理用タイトル |
| body | TEXT | 投稿文 |
| sns_target | VARCHAR(50) | 投稿先（x / instagram / facebook / note / other） |
| status | VARCHAR(20) | draft / posted |
| author_id | BIGINT UNSIGNED | 作成者のWPユーザーID |
| posted_at | DATETIME NULL | 実際に投稿した日時 |
| created_at | DATETIME | 作成日時 |
| updated_at | DATETIME | 更新日時 |

### wp_snapbaton_post_set_images（投稿セット×画像）

| カラム | 型 | 説明 |
|:--|:--|:--|
| post_set_id | BIGINT UNSIGNED | FK → post_sets.id |
| image_id | BIGINT UNSIGNED | FK → images.id |
| sort_order | INT | 投稿セット内の画像順 |

## wp_options に保存する設定値

| オプション名 | 型 | 説明 |
|:--|:--|:--|
| snapbaton_upload_passcode | string | 公開アップロードページの4桁パスコード |

## REST APIエンドポイント

### 公開アップロード用

```
GET  /wp-json/snapbaton/v1/upload-page         → アップロードページHTML返却
POST /wp-json/snapbaton/v1/upload-auth          → パスコード認証（トークン発行）
GET  /wp-json/snapbaton/v1/upload-groups        → グループ一覧（トークン認証）
POST /wp-json/snapbaton/v1/upload-groups        → グループ新規作成（トークン認証）
POST /wp-json/snapbaton/v1/upload-files/{group_id} → ファイルアップロード（トークン認証）
```

### 共有URL用（既存）

```
GET /wp-json/snapbaton/v1/share/{image_id}
```

レスポンス例:
```json
{
  "title": "現場の組子細工",
  "description": "杉材で組んだ麻の葉模様。光の透け方が美しい。",
  "image_url": "https://example.com/wp-content/uploads/2026/03/kumiko.jpg",
  "group": "藤田邸リフォーム",
  "tags": ["組子", "杉", "リフォーム"]
}
```

AIがこのURLを読み取り、画像と説明文を把握してSNS投稿文を生成できる。
