# 02. 機能仕様

## A. ギャラリー機能（基盤）

### グループ管理

| 機能 | 説明 |
|:--|:--|
| グループ作成 | 案件名・説明・タグを入力して作成 |
| グループ編集 | 名前・説明・タグの変更 |
| グループ削除（ゴミ箱） | グループごとゴミ箱へ移動（中の画像も含む） |
| タグ分類 | タグでグループをフィルタリング |

#### グループの属性
- グループ名（必須）
- 説明文
- 作成日（自動）
- タグ（複数可）

### 画像管理

| 機能 | 説明 |
|:--|:--|
| 複数画像・動画アップロード | オリジナルサイズを保持。画像・動画の両方に対応 |
| 画像一覧表示 | グリッド表示 / リスト表示の切替 |
| タイトル入力 | 各画像にタイトルを設定（フォーカス外れで自動保存） |
| 説明文入力 | 各画像に説明文を設定（フォーカス外れで自動保存） |
| 画像タグ付け | 各画像にタグを複数付与（グループタグと共通のタグマスタを使用） |
| EXIF読み取り | アップロード時にEXIF内の説明を自動取得 |
| EXIF書き込み | 説明文をEXIFに書き戻し |
| オリジナルサイズダウンロード | 素材管理用途。元画像をそのまま取得 |
| 画像個別編集 | `@fujiruki/react-image-editor-lightbox` を使用（検討中） |
| 画像個別削除（ゴミ箱） | 画像をゴミ箱へ移動 |
| 画像の並べ替え | ドラッグ&ドロップで表示順を変更 |

### タグ入力UI仕様

グループ・画像の両方で共通のタグ入力UIを使用する。

| 要素 | 説明 |
|:--|:--|
| 付与済みタグ表示 | タグをバッジ表示。×ボタンで個別削除 |
| 最近使ったタグ | 直近で使われたタグ10件をボタンとして表示。ワンタップで追加 |
| 「その他」ボタン | クリックするとコンボボックス（検索付きドロップダウン）を表示 |
| コンボボックス検索 | 既存タグをインクリメンタルサーチで絞り込み。選択で追加 |
| 新規タグ作成 | コンボボックスで一致するタグがない場合、入力文字列で新規タグを作成 |

タグは `wp_snapbaton_tags` テーブルで一元管理し、グループ・画像の両方から参照する。

## D. スマホ公開アップロード機能

### 概要
WPログイン不要で、スマホからURLにアクセスするだけで画像・動画をアップロードできる軽量ページ。
現場の撮影者（職人さん等）にURLを送って「ここに写真上げて」と使う。

### 仕様

| 項目 | 説明 |
|:--|:--|
| アクセスURL | `https://door-fujita.com/wp-json/snapbaton/v1/upload-page` |
| 認証方式 | 4桁パスコード（WP管理画面の設定で変更可能） |
| グループ選択 | 既存グループの一覧から選択、または新規グループ名を入力して作成 |
| アップロード | 複数の画像・動画を一括選択してアップロード |
| スマホ最適化 | モバイルファーストのレスポンシブUI |
| ホーム画面対応 | PWAマニフェスト付きでスマホホーム画面にショートカット追加可能 |

### セキュリティ
- 4桁パスコードはブラウザのSessionStorageに保持（ブラウザを閉じるまで有効）
- パスコードはWP管理画面の SnapBaton設定 で変更可能（デフォルト: 管理者が初期設定時に指定）
- アップロード先はSnapBatonのグループのみ（WP本体への影響なし）

### ゴミ箱

| 機能 | 説明 |
|:--|:--|
| ゴミ箱一覧 | 削除済みの画像・グループを表示 |
| 復元 | ゴミ箱から元の場所に戻す |
| 完全削除 | ストレージから完全に削除 |

### エクスポート

| 機能 | 説明 |
|:--|:--|
| CSVエクスポート | 画像ファイル名・タイトル・説明文の一覧をCSV出力 |

## B. SNSワークフロー機能（拡張）

### 投稿セット

| 機能 | 説明 |
|:--|:--|
| 投稿セット作成 | 投稿に使う画像を1枚以上選択し、投稿文を作成 |
| 画像選択 | グループ内から投稿に使う画像を選ぶ |
| 画像順番入れ替え | ドラッグ&ドロップで投稿画像の順番を変更 |
| 投稿文入力 | SNS投稿用のテキストを入力 |
| 投稿先SNS選択 | X / Instagram / Facebook / note / その他 |
| ステータス管理 | 下書き → 投稿済みなどの状態管理 |

### 投稿記録

| 機能 | 説明 |
|:--|:--|
| 投稿履歴保存 | 「この画像でこんな投稿をした」という記録を保存 |
| 投稿履歴一覧 | 画像ごと/グループごとに過去の投稿を一覧表示 |

### クリップボード連携

| 機能 | 説明 |
|:--|:--|
| 説明文コピー | 画像の説明文をクリップボードにコピー |
| 投稿文コピー | 投稿セットの投稿文をクリップボードにコピー |
| 画像コピー | 画像をクリップボードにコピー |
| 共有URLコピー | 画像と説明文が読み取れるURLをクリップボードにコピー |

#### 共有URLの用途
AIに画像と説明文をまとめて伝えるためのURL。
例: `https://example.com/wp-json/snapbaton/v1/share/{image_id}`
AIがこのURLを読み取り、SNS投稿文を生成できる。
