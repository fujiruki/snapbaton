# 01. 概要

## プロジェクトの目的
写真撮影者とSNS投稿担当者をつなぐWordPressプラグイン。
ギャラリー機能（自社素材管理）とSNS投稿ワークフロー機能を統合する。

## 解決する課題
- 撮影した写真をSNS担当に渡すとき、どの写真に何を書けばいいか伝わりにくい
- 写真が増えると管理が煩雑。案件・現場ごとに整理したい
- 過去にどの画像でどんなSNS投稿をしたか記録が残らない
- AI（Claude等）にSNS投稿文の作成を依頼するとき、画像と説明を伝えるのが手間

## 対象ユーザーと役割（WPロールにマッピング）

| 役割 | WPロール | 操作内容 |
|:--|:--|:--|
| コンテンツ管理者 | 管理者 | 全操作（アップロード・編集・削除・設定） |
| 撮影者 | 編集者 or 投稿者 | 写真アップロード、グループ作成、タイトル・説明文入力 |
| SNS投稿担当者 | 編集者 | 投稿文の作成・編集、画像選択・並べ替え、CSVエクスポート |
| 閲覧者 | 購読者 | 写真と説明文の閲覧のみ |

## ユースケース

### 基本フロー
1. 撮影者がスマホ/PCで現場写真を撮影
2. WP管理画面からグループ（案件）を作成し、写真を複数アップロード（オリジナルサイズ）
3. 各写真にタイトル・説明文を入力
4. SNS投稿担当者がWP管理画面で写真と説明文を確認
5. 投稿したい画像を選んで順番を入れ替え、投稿文を作成
6. X/Instagram/Facebook/note等にコピペで投稿（またはAI経由で投稿）

### AI連携フロー
1. 画像と説明文の「共有URL」をクリップボードにコピー
2. AIに「この内容でTwitter投稿を作って」と依頼
3. AIが投稿文を生成 → ブラウザ操作で投稿

## プラグインの2層構造

| 層 | 機能 | 単体でも価値がある |
|:--|:--|:--|
| 基盤：ギャラリー | アルバム管理、高画質アップロード、オリジナルDL、EXIF、表示 | ○ |
| 拡張：SNSワークフロー | 投稿セット作成、画像選択・並べ替え、投稿文・投稿記録、クリップボード連携 | ○ |

## SNS投稿先
- X（Twitter）
- Instagram
- Facebook
- note
- その他（汎用的に対応可能な設計）
