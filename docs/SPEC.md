# SnapBaton — 仕様目次

## 概要
WordPress ギャラリー & SNSワークフロープラグイン。
高画質な写真ギャラリーに、撮影者→SNS投稿担当者への連携機能を搭載。
撮影者が写真と説明をアップ → SNS担当が投稿文を作成 → X/Instagram/Facebook/note等へ投稿。
